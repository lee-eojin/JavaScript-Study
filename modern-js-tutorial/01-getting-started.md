# 01. Getting Started

## 1-intro: 자바스크립트란?

자바스크립트가 원래 LiveScript라는 이름이었다가, 당시 자바 인기에 편승하려고
이름을 바꿨다는 게 좀 웃기다. 덕분에 아직도 자바랑 헷갈리는 사람들이 있는데,
실제로는 아무 관련 없는 완전히 다른 언어다.

브라우저마다 자바스크립트 엔진이 다르다는 것도 알게 된다. Chrome은 V8,
Firefox는 SpiderMonkey를 쓴다. 나중에 "V8에서만 지원" 같은 문구를 보면
Chrome/Opera 한정이라는 뜻이니까 기억해둬야겠다.

> **파싱(Parsing)과 컴파일(Compile)**
>
> 엔진이 코드를 실행하는 과정:
> 1. 파싱: 코드를 읽어서 문법 구조를 분석 (사람이 쓴 코드 → 컴퓨터가 이해할 수 있는 구조)
> 2. 컴파일: 파싱된 결과를 기계어로 변환
> 3. 실행: 변환된 기계어 실행
>
> 자바스크립트는 인터프리터 언어라고 알고 있었는데, 실제로는 JIT(Just-In-Time)
> 컴파일을 해서 실행 중에도 최적화를 계속 한다고 한다.

### 브라우저에서 할 수 있는 것 vs 없는 것

할 수 있는 것:
- HTML/CSS 조작
- 사용자 이벤트 반응 (클릭, 키보드 등)
- 서버 통신 (AJAX)
- 쿠키, 로컬 스토리지 저장

못하는 것 (보안 제약):
- 사용자 PC의 임의 파일 읽기/쓰기
- 다른 탭/창의 정보 접근 (동일 출처 정책)
- 카메라/마이크 무단 접근

> **동일 출처 정책 (Same Origin Policy)**
>
> 프로토콜, 도메인, 포트가 모두 같아야 "같은 출처"로 인정된다.
> `http://site.com`에서 `http://other.com`의 데이터를 마음대로 가져올 수 없다.
>
> 나중에 API 호출할 때 CORS 에러 만나면 이 개념이 원인이다.
> 서버에서 허용해줘야 크로스 도메인 요청이 가능하다.

### 트랜스파일 언어들

자바스크립트로 변환되는 언어들이 있다. TypeScript가 대표적인데,
타입을 명시해서 런타임 에러를 미리 잡을 수 있다.
결국 브라우저는 자바스크립트만 이해하니까, 이런 언어들도 결국 JS로 변환된다.

> **트랜스파일(Transpile)**
>
> 컴파일이 고수준 → 저수준(기계어) 변환이라면,
> 트랜스파일은 고수준 → 고수준 변환이다. (TypeScript → JavaScript)

## 2-manuals-specifications: 매뉴얼과 명세서

자바스크립트 공부하다 막히면 어디서 찾아봐야 하는지에 대한 내용이다.

### ECMA-262 명세서

자바스크립트의 공식 명세서다. 언어 자체를 정의하는 문서인데,
형식이 딱딱해서 일상적인 참고 자료로는 별로라고 한다.
새로운 기능이 추가되기 전에 어떤 제안들이 있는지는
[tc39/proposals](https://github.com/tc39/proposals)에서 볼 수 있다.

> **Stage 3?**
>
> TC39에서 새 기능을 추가할 때 Stage 0~4 단계를 거친다.
> Stage 3이면 거의 확정된 상태라서, 곧 정식으로 들어올 기능이라고 보면 된다.

### MDN

실질적으로 가장 많이 쓰게 될 사이트다.
특정 함수나 메서드 찾을 때 구글에 `MDN parseInt` 이런 식으로 검색하면 바로 나온다.

> **실전 팁**
>
> 모르는 메서드 나오면 일단 "MDN [메서드명]"으로 검색하는 습관을 들이면 좋다.
> 예제 코드도 같이 나와서 이해하기 쉽다.

### 호환성 확인

브라우저마다 지원하는 기능이 다르니까, 내가 쓰려는 기능이 되는지 확인해야 할 때가 있다.

- [caniuse.com](https://caniuse.com) - 브라우저별 지원 여부 표로 보여줌
- [kangax compat-table](https://kangax.github.io/compat-table) - 엔진별 기능 지원 여부

> **언제 쓰나?**
>
> IE 지원해야 하는 프로젝트거나, 최신 문법 쓰고 싶은데 될지 모를 때 확인한다.
> 요즘은 IE 지원 안 하는 곳이 많아서 자주 볼 일은 없을 수도 있지만,
> 회사마다 다르니까 알아두면 좋다.
